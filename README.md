##  Notes
- This website uses Jekyll. Go to the homepage to learn more: http://jekyllrb.com/docs/home/

- All Styles are generates using Bootstrap and a few custom styles in styles/styles.css

- Older/unused files are in /.etc

##  Instructions

1. Install Jekyll
  		gem install jekyll
2. Run server
		jekyll serve
3. local url
		http://localhost:4000/
